Introduction

This document is a guide to creating an embroidery file from existing graphics using Ink/Stitch. To open different sections of the guide, utilize the buttons in the top-right. Browse the sections from left to right and follow the instructions carefully.

Tip: If any problem is encountered in the process, please refer to the official user mannual of Ink/Stitch, especially the workflow section. There is also a YouTube tutorial demonstrating the process by the channel Low Tech Linux.

1Installation

1.1 Acquire installation files

It is necessary to install Inkscape for using Ink/Stitch, as Ink/Stitch is an extension of it. To acquire the respective installation file, visit the installation page of Inkscape and Ink/Stitch. Ensure that the installation file is built for the operating system you are running.

Warning: When installing on macOS, ensure the installation file is built for your hardware architecture. In addition, ensure that Inkscape is installed and run before installing Ink/Stitch. Otherwise, the installation will fail.

1.2 Install Inkscape

Windows
Run the installation file and follow the instructions displayed by the installer.

macOS
Run the installation file, then drag the Inkscape icon into the Applications folder.

1.3 Install Ink/Stitch

Windows
Run the installation file. A window will be displayed with the text “Windows protected your PC”. This is due to the windows certificate of Ink/Stitch not having enough trust. To continue the installation, select “more info” in the pop-up window, then select “run anyway”. A setup window will be displayed and guide you through the installation process, in which you do not have to make any changes to the default settings.

macOS
Run the installation file. A setup window will be displayed and guide you through the installation process, in which you do not have to make any changes to the default settings.

2Import

2.1 Prepare source graphic

Prepare the graphic which you want to embroider in JPEG or PNG format. Ways to achieve this include using online sites such as online-convert.com, or using graphic editing softwares such as GIMP.

2.2 Create a vector file

Vector files are recommended because Ink/Stitch can convert vector files directly to embroidery files. To convert your graphic into a vector format, use online sites such as pngtosvg.com or a graphic editing software.

Warning: When converting to a vector file, ensure that the converted vector file includes the colors of the source graphic, as some online sites omit the colors. In this case, use the ways for conversion that are stated previously.

2.3 Import to Inkscape

Open the vector file with Inkscape by right clicking it in your file manager, then select Open with > Inkscape.

3Adjust

Tip: Images are linked at the end of each step to demonstrate the step.

3.1 Open Layers and Objects panel

From the top menu, select Object > Layers and Objects. A panel will display on the right of the Inkscape window. Expand the folder inside the panel by clicking the triangle icon next to it.
3.1

Tip: If the graphic is too small, select the magnifying glass icon in the bottom left of the Inkscape window, then click on where you want to zoom in.

3.2 Delete layers

Before embroidering, you may want to delete any layers which you do not want to embroider. To see a layer’s effect on the graphic, hover over it in Layers and Objects panel, then click the eye icon next to the cursor to hide or unhide the layer. To delete a layer, select it in the panel and click the trash can icon in the top right.
3.2

3.3 Adjust colors

Before embroidering, you may want to adjust the colors of the graphic. To do so, select the layer which you want to change the color of and select one of the colors at the bottom of the Inkscape window. If you want to use a custom color, open Fill and Stroke panel by selecting Object > Fill and Stroke from the top menu. Then select the layer and input the hexadecimal RGBA value of the color into the RGBA box in the bottom right of the panel.
3.3

Warning: Ink/Stitch will not create a stitch plan with colors if your graphic includes color gradients. In this case, replace the color gradient with another color.

3.4 Preview the stitch plan

To preview the stitch plan, select Extensions > Ink/Stitch > Visualize and Export > Simulator / Realistic Preview. If you wish to make any adjustments after viewing the stitch plan, go back to step 3.2 and 3.3.

4Export

4.1 Export embroidery file

To export the embroidery file, select File > Save As from the top menu. In the pop-up window, click the button with the text “Inkscape SVG”, then select an embroidery format from the list of file types. Additionally, you can change the location the file is saved at by selecting a folder in the left of the window. Finally, click the “Save” button to save it on your machine.
